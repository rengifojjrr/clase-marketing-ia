ChatGPT — Investigación Profunda

Usamos ChatGPT para que investigue por nosotros. Le pedimos que busque negocios de un nicho específico en nuestra ciudad que tengan buenas reseñas pero no tengan página web. En minutos tenemos una lista completa de prospectos.
¿Por qué usar ChatGPT primero?
✓ Busca en segundos lo que tomaría horas
✓ Analiza reseñas y presencia online
✓ Genera una lista organizada
✓ Encuentra patrones del mercado
✓ Sugiere cuáles son más rentables
Lo que le pedimos a ChatGPT:
📍 Ciudad o zona específica
🏪 Tipo de negocio (nicho)
⭐ Con 4+ estrellas en Google
🚫 Sin página web propia
📱 Solo con redes sociales
💡 Luego verificamos manualmente en Google Maps los negocios que ChatGPT nos da, para confirmar que realmente no tienen web y ver su perfil completo.
Prompt · ChatGPT

Actúa como un investigador de mercado especializado en negocios locales.
Necesito que busques prospectos para mi servicio de páginas web con IA.

LO QUE BUSCO:
Negocios de [TIPO DE NEGOCIO: restaurantes / barberías / spas / gimnasios / etc.]
ubicados en [CIUDAD O ZONA] que cumplan TODAS estas condiciones:

✓ Tienen 4 estrellas o más en Google Maps
✓ Tienen al menos 20 reseñas de clientes
✓ NO tienen página web propia (solo tienen redes sociales)
✓ Están activos (publicaron contenido en los últimos 3 meses)

PARA CADA NEGOCIO QUE ENCUENTRES, DIME:
1. Nombre del negocio
2. Dirección o zona de la ciudad
3. Número de estrellas y cantidad de reseñas

5. Por qué crees que necesita una página web
6. Qué tipo de página web le iría mejor

FORMATO DE RESPUESTA:
Organiza los resultados en una tabla.
Ordénalos de mayor a menor potencial de venta.
Incluye al menos 10 negocios.

ANÁLISIS FINAL:
Al terminar la lista, dime:
- Cuál es el nicho con más oportunidades en esa zona
- Cuál sería el mensaje de apertura ideal para contactarlos
- Cuánto podrías cobrar en ese mercado específico

Usa tu capacidad de búsqueda web para encontrar información real y actualizada.

1

3
Lee las reseñas de Google Maps
Si 8 de 10 reseñas dicen "el pollo asado es increíble" — ese es el producto estrella sin dudas.
💡 Copia 3 reseñas reales positivas — las usarás como testimonios en la página

Este es el input para Google Flow (el video) y para Gemini (los textos). Tenlo listo.

🎬 Si el video no reproduce en móvil
Usar cuando el video no hace autoplay en iPhone o Android · Copia y pega en Lovable o Bolt
The hero video is not autoplaying on mobile devices. Fix it completely.

1. ADD ALL FOUR REQUIRED ATTRIBUTES:
The video element must have autoplay, muted, loop and playsinline all set
at the same time. Without all four together the video will not autoplay
on iPhone or Android.

2. USE POSITION FIXED FOR THE SCROLL EFFECT:
Set the video element to position fixed, top 0, left 0, width 100vw,
height 100vh, object-fit cover, z-index 0. This locks the video in the
background at all times.

Place a dark overlay div with the same position fixed, background
rgba(0,0,0,0.5), z-index 1.

The hero content div must be position relative, height 100vh, z-index 2
so it sits above the overlay.

Every section below the hero must have position relative, z-index 2, and
a solid background color so those sections cover the fixed video as the
user scrolls down the page.

3. DO NOT USE background-attachment fixed:
background-attachment: fixed does not work on mobile browsers.
Use position fixed on the video element as described above —
it works on all devices including iOS Safari and Android Chrome.

Fix this without changing anything else. Do not ask questions.
